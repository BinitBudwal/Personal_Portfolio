HTML

HTML stands for Hyper Text Markup Language, which is considered as the basic building block of the web pages and web as a whole. Main purpose of HTML is to provide structure, describe a page’s apparel and to provide a place to add functionality.

CSS

CSS stands for Cascading Style Sheets, a styling language used to describe how HTML elements should be shown on request. Along with HTML, CSS is also a core component of the web. CSS provides with styles and layout of the web page and can alter the Font, colour, size and spacing of the content. It does so with CSS properties which target the HTML element’s style aspect

LikeHTML, CSS has many components to itself like CSS rule, Selector and value.

CSS Rule - It is a grouping of one or more CSS properties that are applied to the targeted HTML element.

CSS Selector - A part of CSS rule that determines what HTML element is affected. There are many different types of Selectors Like Universal Selector, Element Selector, Class/Id selector, attribute selectors and a few more less important.

CSS Values - These are the values that are set against the CSS properties that declare the value of the content.
